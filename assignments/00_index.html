Computer Graphics, Spring 2022

Here is a collection of projects I put together for my computer graphics course in the spring of 2022

  1. Drawing a Circle: Learn how to work with HTML, Javascript and the <canvas> element.
  2. Graphing Calculator: Create a graphing calculator, learn how to convert between spaces with lerp.
  3. Orthographic Projection: Create a basic 3d grapher, apply rotation matrices, work with indexed geometry.
  4. Perspective Projection: Update your grapher to use perspective projection
  5. Color: Learn about various representations of color and how to draw quickly to a canvas.
  6. Dot and Cross Products: Learn about dot products, cross products, and linear combinations and their applications in computer graphics.
  7. Backface Culling: Apply what we know about dot and cross products to determine which faces of objects to draw.
  8. Z-buffers: By drawing our faces pixel by pixel and keeping track of the depths of those pixels we can get an even more accurate 3d model.
  9. Textures: We can use the u and v coordinates calculated in the previous assignment to apply textures to our objects.
  10. Pipeline: Comparing our 3d renderer to the graphics pipeline
  11. THREE.js: Now that we know how graphics rendering works, we'd like to use our graphics cards to speed up the process. We will be using WebGL: a version of OpenGL for the web, and THREE.js: a wrapper library to make working with WebGL easier.
  12. Spaces: Converting from model coordinates to world space to camera space involves a number of matrix operations and translations. We introduce a fourth coordinate set to 1 so these operations are just matrix operations and explore how these matrices are set up and composed.
  13. Procedural Geometry: We learn how to create our own geometry, specifying vertex coordinates and additional attributes for vertices.
  14. Shaders: Shaders allow us to execute arbitrary code in a language called GLSL at two key stages of the graphics pipeline.
  15. Uniforms: Uniforms allow us to pass data to our shaders, including slider values, the current time, and texture data.
  16. Lighting: All modern graphical effects are achieved with shaders. In this assignment, we make our own lighting shaders.
  17. Above and Beyond: A list of some advanced graphics techniques to look into and resources for learning more about computer graphics.
